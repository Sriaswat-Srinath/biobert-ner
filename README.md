# BioBert NER Project

This repository contains the code and data for a Biomedical Named Entity Recognition (NER) system using BioBERT.

## Structure

* `src/`: Source code for the project.
  * `src/ner_model/`: Scripts for training and deploying the NER model.
  * `src/big_data_pipeline/`: Data ingestion pipeline (e.g., Kafka producer).
  * `src/spark_poc/`: Apache Spark proofs-of-concept for data processing.
  * `src/demo/`: Streamlit application for inference demonstration.
* `data/`: Raw and processed data used for model training and evaluation.
* `notebooks/`: Jupyter notebooks for exploratory data analysis and demonstrations.
